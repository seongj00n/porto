PROJECT 01

KYOBO BOOK

Open API, JSON

POINT COLOR

국내 도서 사이트인 교보문고의 웹페이지입니다.

Kakao에서 제공하는 Open API 데이터를 받아 AJAX를 사용하여
도서 정보들을 나타냈습니다.

소비자들이 상품 이미지에 집중할 수 있도록 배경을 흰색으로 하였고 로고의 컬러를 포인트 컬러로 사용하였습니다.

jQuery를 사용하여 슬라이더, 배너 등을 구현하여 사용자들의 편의성을 높혔습니다.

PROJECT 02

NeoBeat

Responsive Website

POINT COLOR

NeoBeat 를 홍보하는 웹페이지입니다.

화려하고 활동적인 느낌를 전달하는 분홍색과 예술 및 창의력과 독창적 에너지를 상징하는 보라색을 포인트 컬러로 사용하였습니다.

사용자가 엑스포 개최일까지 남은 기간을 확인할 수 있도록 자바스크립트로 구현하였습니다.

디스플레이 종류에 따라 화면의 크기가 자동으로
최적화 되도록 반응형 웹페이지로 제작하였습니다.

Shopping Mall
